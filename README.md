# Hospital-Admissions-Excel-Analysis
Excel dashboard analyzing hospital emergency room admissions, patient demographics, wait times, and satisfaction scores.
Overview
This project presents an Emergency Room Dashboard built in Excel to analyze hospital admission data. It highlights patient flow, satisfaction scores, wait times, and departmental referrals. The dashboard uses pivot tables, charts, and formulas to transform raw data into meaningful insights for healthcare management.

Objectives
- Track patient admissions and discharge status
- Monitor average wait times and satisfaction scores
- Analyze demographics (age, gender) and referral departments
- Provide actionable insights for hospital resource planning

Tools & Skills Used
- Excel: Pivot tables, charts, formulas
- Dashboard design and visualization
- Data cleaning and structuring

Key Insights
- 53% of patients admitted, 47% not admitted
- Average wait time: ~34 minutes
- Female patients slightly higher (53%) than male (47%)
- Orthopedics and General Practice are top referral departments
- Peak admissions in age group 40–49

How to use
- Download the `.xsls` file from this repository
- Open in Excel
- Navigate through the dashboard tabs to explore patient trends and metrics
